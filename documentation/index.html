Protected Endpoints

All /user, /admin, /token-manager endpoints are protected. To access these endpoints, you must provide the following request headers:

  • Authorization: Bearer token
  • X-Custom-loginMethod: "google" or "normal"
  • X-Custom-role: "user" or "admin"

Error-Throwing Queries

The schema is very strict and will throw an error if the requested data is either not found or the provided data is incorrect.
